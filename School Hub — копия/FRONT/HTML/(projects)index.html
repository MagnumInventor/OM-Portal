Глобальні проєкти

Сила таланту (напрям проектів)

Мета проєкту — реалізація проєктів для розвитку та демонстрації спортивних, творчих та виробничих здібностей учнів. Заплановано проведення таких заходів, як:

  • Онлайн маркетплейс — дасть можливість будь-якому учню виставляти продукт власного виробництва на продаж.
  • Проведення спортивних змагань — підтримка фізичної активності та здорового способу життя серед учнів.
  • Мистецька галерея — платформа для демонстрації творчих робіт учнів.
  • ФотоКлуб — розвиток навичок відео/фотозйомки.

Учнівська АкадеміЯ

Мета проєкту — створення платформи для всебічного розвитку учнів через систему учнівського репетиторства, профорієнтаційні лекції та підготовку до НМТ/IELTS. Доступ буде надаватися у вигляді закритої спільноти з місячною підпискою. Також плануються:

  • Репетиторство (для 5-9 класів): Раз на тиждень з таких предметів, як математика, українська мова, англійська мова, хімія.
  • Профорієнтація: Лекції для учнів, що допоможуть визначитися з майбутньою професією.
  • Міжнародні поїдки та тури по Україні Раз на 2 місяцi (залежно від спонсорів та дирекції).

Допоміжні ініціативи

Глобальні перспективи

Дослідження міжнародних програм, міграції та життя за кордоном аналізує ключові аспекти, пов'язані з можливостями та викликами для людей, які розглядають переїзд до інших країн. Це включає оцінку фінансових витрат (вартість життя, оренда житла), економічні перспективи (середні зарплати, податкові ставки), а також тривалість процесу отримання громадянства та адаптації. Міграційні програми, такі як "Green Card" або резидентство в Канаді, дозволяють краще зрозуміти умови імміграції.
